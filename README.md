# dotfiles

## Installation

```bash
~$ git clone https://github.com/initbar/dotfiles.git ~/.lib
~$ cd ~/.lib && git submodule update --init --recursive
```

## Docker

```bash
~$ # docker pull initbar/dotfiles:latest
~$ docker run -it --rm initbar/dotfiles:latest /bin/bash
```
