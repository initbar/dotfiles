# lib

## Installation

```bash
~$ git clone https://gitlab.com/initbar/lib.git ~/.lib
~$ cd ~/.lib && git submodule update --init --recursive
```

## Docker

```bash
~$ # docker pull docker.init.bar/initbar/lib:latest
~$ docker run -it --rm docker.init.bar/initbar/lib:latest /bin/bash
```
