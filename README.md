# lib

## Installation

```bash
~$ git clone https://github.com/initbar/lib.git ~/.lib
~$ cd ~/.lib && git submodule update --init --recursive
```

## Docker

```bash
~$ # docker pull initbar/lib:latest
~$ docker run -it --rm initbar/lib:latest /bin/bash
```
